# Journal.

A minimal blog about design, code, and craft.

Built with intention and care.

---

## Stack

- [Next.js](https://nextjs.org) — framework
- [Tailwind CSS](https://tailwindcss.com) — styling
- [Newsreader](https://fonts.google.com/specimen/Newsreader) + [Inter](https://rsms.me/inter/) — typography

## Running locally

```bash
pnpm install
pnpm dev
```

## Writing

Posts live in `lib/posts.ts`. Each entry is a title, date, category, excerpt, and an array of paragraphs. Add one there and it appears everywhere.
